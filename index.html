🌐 网络质量测试工具

功能全面的网络诊断与监控工具，支持Ping测试、速度测试、流量统计、局域网扫描等功能

Python 3.8+ PyQt5 跨平台 开源免费

功能特性

🔍
网络接口检测
自动检测网络接口信息，包括IP地址、MAC地址、信号强度、WiFi名称等
📡
Ping测试
可配置目标地址和测试次数，显示平均时延和丢包率统计
⚡
速度测试
下载/上传速度测试，支持speedtest-cli和curl两种测速方式
📊
持续测速监控
定时自动测速并记录历史数据，生成统计图表
📈
流量统计
实时监控网络流量，支持按日/周/月查看历史统计
🛤️
路由追踪
追踪网络路由路径，显示每一跳的IP和延迟
🌐
局域网扫描
扫描局域网内所有设备，显示IP、MAC地址和延迟
🔍
DNS解析测试
测试DNS服务器解析速度和正确性

下载安装

🐧
Linux (Ubuntu/Debian)
network-tool_1.0.0_amd64.deb
下载 .deb 包 安装说明
1
安装deb包
sudo dpkg -i network-tool_1.0.0_amd64.deb
2
安装依赖（如有缺失）
sudo apt-get install -f
3
运行程序
network-tool
🪟
Windows
NetworkTool.exe (需自行打包)
打包说明
1
安装Python和PyQt5
pip install PyQt5 pyinstaller
2
使用spec文件打包
pyinstaller network_tool.spec
3
运行生成的exe
dist\NetworkTool.exe

依赖要求

依赖项 Linux Windows
Python python3 (>=3.8) Python 3.8+
GUI框架 python3-pyqt5 PyQt5
速度测试 speedtest-cli, curl speedtest-cli
Ping测试 iputils-ping 系统自带
路由追踪 traceroute 无需额外依赖
DNS工具 dnsutils 无需额外依赖
网卡信息 ethtool, iwconfig 无需额外依赖

快速使用

1
检测网络接口 - 点击"🔍 检测网络"按钮，选择要测试的网络接口
2
Ping测试 - 输入目标IP或域名，设置测试次数，点击"📡 Ping"
3
速度测试 - 点击"⚡ 速度测试"或"🚀 全部测试"进行测速
4
流量监控 - 点击"▶️ 流量监控"开始实时流量统计
